# Whitepaper

Dogechain is an EVM-compatible blockchain that aims to complement the original Dogecoin cryptocurrency. As a proof-of-stake blockchain, Dogechain seeks to bring scalability, security, robustness, and utility to Dogecoin. In short, Dogechain doesnâ€™t compete with Dogecoin.
